Image Processing Automation System
Project Objective
Automate the image processing workflow for real estate property listings.
The goal of Phase 1 is to generate WordPress-ready image assets from raw photo and THETA image files while keeping image selection, image ordering, and final publishing under manual staff control.

Current Workflow

1. Select property from Reapro..
2. Create draft post in WordPress..
3. Edit property information..
4. Take property photos and videos..
5. Retouch and rename images..
6. Create floor plan..
7. Create video..
8. Upload images to WordPress..
9. Verify THETA placement..
10. Add YouTube URL..
11. Select images and thumbnail..
12. Final review and publish..

Scope of Phase 1
Phase 1 covers only:

• Photo processing.
• THETA image processing.
• File organization.
• File renaming.
• Image retouching.
• Logo insertion.
• Export of completed image files.
WordPress upload, YouTube integration, and publishing workflow are excluded from Phase 1.
